Mi solución a la actividad 5.4 ha sido:

El archivo /etc/fstab contiene la información de los puntos de
montaje, en este caso hay 6 puntos de montaje, aunque solo hay
uno "físico" que es / los demás son todos temporales para
/dev/pts para /tmp, para /proc, etc… las opciones que usa son
las comunes, el auto para / que detecta automáticamente el
sistema de archivos, pero se podría especificar, / tiene el
atributo noatime que haee que el sistema de archivos no escriba
la hora del ultimo acceso a un archivo, esto mejora el
rendimiento en discos SSD y está indicado para discos
virtuales, como es el caso de User-Mode Linux
